SCRIPTING WINDOWS POWERSHELL

#StartOrStopService()
Il vous est demandé de réaliser un script « StartStopAService.pws » qui permet de démarrer 
ou stopper n’importe quel service.
Le script reçoit donc comme paramètres le nom du service et l’action start ou stop.

#ADNewUser()
1 Il vous est demandé de réaliser un script Powershell « AddViewUsers.pws » qui reçoit 
les paramètres suivants : utilisateur, mot de passe, domaine Le script ajoute l’utilisateur 
sur le domaine spécifié.
2 Si seul le paramètre « /liste » est fourni, la liste des utilisateurs du domaine est 
affichée.
